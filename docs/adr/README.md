# Architecture Decision Records

Directorio de ADRs. Una decisión no trivial = un archivo `NNNN-titulo-corto.md`.

## Formato

```markdown
# NNNN — Título corto

**Fecha:** YYYY-MM-DD
**Estado:** Propuesto | Aceptado | Rechazado | Deprecado | Superado por NNNN

## Contexto
Qué problema estamos resolviendo y por qué ahora.

## Decisión
Qué decidimos hacer.

## Alternativas consideradas
Qué otras opciones evaluamos y por qué las descartamos.

## Consecuencias
Qué cambia (bueno y malo) como resultado.
```
